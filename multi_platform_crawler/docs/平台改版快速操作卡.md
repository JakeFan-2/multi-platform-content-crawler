# 平台改版快速操作卡

面向运维：页面结构变化时，优先改**外置 YAML**；仅当登录流程、翻页逻辑等代码级行为变化时才改**采集器 `.py`**。修改任意外置文件后，**必须完全退出 exe 再重新启动**方能生效。

---

## 1. 如何修改 YAML 选择器（示例：`platforms/toutiao.yaml` 的 `row_selectors`）

1. 用 UTF-8 编码的编辑器打开 `platforms/toutiao.yaml`（其他平台则打开 `platforms/<平台ID>.yaml`）。
2. 定位到 `article_list` → `row_selectors`：此处定义**列表里每一行文章卡片/行**的容器选择器，程序会按顺序尝试匹配。
3. **改版后**在浏览器开发者工具中选中新的文章行节点，将**更稳定、更靠前**的 CSS 写在列表**最上面**（与采集器内「先试第一个选择器」的逻辑一致）。

**摘录示例（头条号，仅作格式参考）：**

```yaml
article_list:
  row_selectors:
    - css: ".article-card-wrap"
    - css: "[class*='article-card-wrap']"
    - css: ".article-card"
    - xpath: "//div[contains(@class, 'article-card-wrap')]"
```

4. 若列表在 iframe 内，还需检查同文件中 `iframe_selectors` 等段落。
5. 保存为 **UTF-8**，关闭编辑器 → **重启 exe** → 再试采集。

---

## 2. 如何替换采集器文件（`collectors/` 目录）

1. 确认平台 ID 与文件名一致，例如 ZAKER 对应 `collectors/zaker.py`。
2. 备份原文件：`collectors/zaker.py` → 如 `zaker.py.bak`。
3. 将新版本 `.py` 覆盖到同路径、**同名**。
4. **完全退出并重新启动 exe**（动态加载在运行期会缓存模块，不重启可能仍用旧逻辑）。

> 搜狐号（`sohu`）已在加载器中禁止加载，请勿替换为依赖搜狐的未审核脚本。

---

## 3. 如何修改 `.env` 中的账号密码

1. 在**与 exe 同级**目录（开发环境为项目根）找到 `.env`。
2. 按平台修改键名，例如 ZAKER：

   ```env
   ZAKER_USERNAME=你的账号
   ZAKER_PASSWORD=你的密码
   ```

3. 飞书相关一般为：`FEISHU_APP_ID`、`FEISHU_APP_SECRET`、`FEISHU_APP_TOKEN`、`FEISHU_TABLE_ID`（以 `utils/feishu_exporter.py` 为准）。
4. 保存后 **重启 exe**。若某平台 YAML 根级仍写有 `username`/`password`，程序可能以环境变量优先；细节见 `utils/env_loader.get_platform_credentials_with_fallback`。

---

## 4. 修改后必须重启 exe

- Python 会缓存已加载模块；`.env`、YAML 均在启动流程或首次读取时生效。
- **正确做法**：退出进程（任务管理器中结束亦可）→ 再次双击 `多平台内容采集系统.exe`。
